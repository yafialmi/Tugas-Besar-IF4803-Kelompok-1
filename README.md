# Tugas-Besar-IF4803-Kelompok-1

## Deskripsi Aplikasi

Ini adalah **Sistem Manajemen Menu dan Pelanggan** yang dikembangkan sebagai tugas besar untuk mata kuliah IF4803. Aplikasi ini mengelola menu restoran/kafe dan informasi pelanggan menggunakan struktur data linked list.

### Fitur Utama:

**Manajemen Menu:**
- Membuat dan mengelola item menu dengan informasi ID, nama, harga, dan stok
- Menyisipkan item menu di awal, akhir, atau setelah item tertentu
- Menghapus item menu dari daftar
- Mencari item menu berdasarkan informasi
- Menampilkan semua item menu dalam sistem

**Manajemen Pelanggan:**
- Membuat dan mengelola profil pelanggan dengan ID, nama, saldo, dan status VIP
- Mempertahankan daftar doubly-linked list pelanggan untuk setiap item menu
- Menyisipkan pelanggan di awal, akhir, atau setelah pelanggan tertentu
- Menghapus pelanggan dari sistem
- Mencari pelanggan berdasarkan informasi
- Menampilkan semua pelanggan yang terkait dengan item menu

### Struktur Data:

- **Linked List (Menu):** Daftar terpaut tunggal untuk menyimpan item menu
- **Doubly-Linked List (Pelanggan):** Daftar terpaut dua arah untuk menyimpan pelanggan setiap item menu
- Struktur hierarki di mana setiap item menu dapat memiliki beberapa pelanggan terkait

### Teknologi:

- **Bahasa:** C++
- **Struktur Data:** Linked Lists (Tunggal dan Dua Arah)
- **Sistem Build:** Code::Blocks (CBP)
